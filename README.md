# Web3-Wallet

## Supported Wallets

-   MetaMask
-   WalletConnect
-   Coinbase Wallet
-   Sequence Wallet
-   Gnosis Safe

## Functions

-   Read address, balance
-   Smart contract interaction
-   Switching between chains
-   Eager connect
-   Sign message
